# Flopsed — Tutorial Drafts

Three versions targeting different audiences and tones. Pick one (or mix & match).

---

## Version A: "The README" (Dry, Minimal, Dev-Friendly)

Assumes the player is tech-literate. No hand-holding. Gets to the point.

---

### How to Play

You're a solo dev in a garage. Type code, sell it, scale a company, build AGI.

**The loop:**
```
Type on keyboard  -->  Lines of Code (LoC)  -->  FLOPS execute them  -->  Cash
     ^                                                                    |
     |______________ buy upgrades, hire devs, unlock AI  <________________|
```

**Three resources:**

| Resource | What it is | How you get it |
|----------|-----------|----------------|
| **LoC** | Code you write | Typing, hiring devs, AI models |
| **FLOPS** | Processing power | Hardware upgrades (CPU, RAM, servers) |
| **Cash** | Money | FLOPS execute your LoC, each line pays |

**The catch:** LoC doesn't become cash directly. FLOPS consume LoC from your queue. No FLOPS = code piles up, no income. Too many FLOPS + no code = idle hardware.

**Progression:** Spend cash in the **Shop** (right panel) on upgrades, and in the **Tech Tree** (center tab) on research nodes. Each tier unlocks new mechanics:

- **Garage** — Just you and a keyboard
- **Freelancing** — Clients pay more per line
- **Startup** — Hire devs who auto-write code
- **Tech Company** — Teams, managers, GPU clusters
- **AI Lab** — AI models write code, but they eat your FLOPS
- **AGI Race** — Exponential scaling toward singularity

**The FLOPS slider** (appears at Tier 4): AI models need FLOPS to generate code, but you also need FLOPS to execute it. The slider splits your FLOPS between the two. Balance it or die.

**Win condition:** Buy "The Singularity" ($500T). Takes ~35-40 minutes.

## Version B: "The Onboarding" (Step-by-step, Friendly, For Muggles)

Written as an in-game tutorial flow. Short sentences. Action-oriented.

---

### Welcome to Flopsed

You're building the next big tech company. Start by typing.

#### Step 1: Type!

Mash your keyboard. Every key you press writes **Lines of Code** (LoC). Watch them appear in the editor on the left.

Your code isn't worth anything yet. It needs to be **executed**.

#### Step 2: Execute = Cash

See that purple number at the top? That's your **FLOPS** — your computer's processing power. FLOPS automatically run your code, turning each line into **cash**.

```
You type  -->  LoC piles up  -->  FLOPS run it  -->  You get paid
```

If you type faster than your FLOPS can process, code stacks up. That's fine — it'll catch up.

#### Step 3: Spend Cash

Open the **Shop** on the right. Buy upgrades:
- **Hardware** (more FLOPS = run code faster)
- **Keyboard upgrades** (more LoC per keystroke)
- **Energy drinks** (type speed boosts)

The more you buy, the faster cash flows. Buy what you can afford, then keep typing.

#### Step 4: Unlock New Tiers

Once you have enough cash, unlock the next **Tier** in the tech tree (center tab). Each tier is a game-changer:

| Tier | What happens |
|------|-------------|
| Freelancing | Your code is worth 2.5x more |
| Startup | You can hire people who write code for you |
| Tech Company | Teams of devs + serious hardware |
| AI Lab | AI writes code — but needs YOUR processing power |
| AGI Race | Exponential chaos. Numbers go brrr. |

#### Step 5: Stop Typing, Start Managing

By Tier 2, you'll have hired devs who auto-generate code. Your job shifts from typing to **buying the right upgrades at the right time**.

By Tier 4, AI models appear. They write code insanely fast, but they consume FLOPS to do it. A **slider** appears — you decide how much processing power goes to:
- **Executing code** (turns LoC into cash)
- **Running AI** (generates more LoC)

Too much execution = AI starves, queue empties. Too much AI = code piles up, no income. Find the sweet spot.

#### Step 6: Build AGI

Keep scaling. Buy bigger models, bigger servers, bigger everything. When you can afford **The Singularity** ($500 trillion), you win.

Total time: about 35-40 minutes. Good luck — the AI is watching.

## Version C: "The Metaphor" (Narrative, Atmospheric, In-Universe)

Written as if the game itself is talking to you. Immersive. The meta-joke baked in.

---

### > ./tutorial.sh

```
Welcome to the grind.

You're sitting in a garage. There's a laptop, a coffee mug,
and an idea that'll change the world. Probably.

Press any key to start coding.
```

**You type. The machine listens.**

Every keystroke produces **Lines of Code**. Raw, unprocessed potential sitting in a queue. It means nothing until a computer runs it.

That's where **FLOPS** come in — your machine's brain. One FLOP processes one line per second. Each processed line becomes **cash**. Cash buys better gear. Better gear means more FLOPS. More FLOPS means more cash. You've just discovered the loop.

```
$ cat ~/.progression

Tier 0: The Garage ............ you + a keyboard
Tier 1: Freelancing ........... clients appear, code is worth more
Tier 2: Startup ............... hire devs, stop typing yourself
Tier 3: Tech Company .......... teams, managers, GPU racks
Tier 4: AI Lab ................ AI writes code. it also eats your FLOPS.
Tier 5: AGI Race .............. exponential. everything. everywhere.
```

**The first pivot** happens when you hire your first developer. Suddenly, code writes itself. You stop typing. You start *managing*. The left panel fills with code you didn't write.

**The second pivot** happens when AI arrives. It's faster than any human — but it's hungry. Your FLOPS, which used to just execute code, now have a second job: feeding the AI. A slider appears. You choose: how much processing power runs code vs. generates it?

```
  Total FLOPS
     |
     |--[====EXECUTION====|===AI===]-->
     |
     slider: 70% exec / 30% AI
```

Slide too far right: AI writes mountains of code nobody runs. No cash.
Slide too far left: Execution waits for code that never comes. Idle.

The slider is the late-game keyboard. Your fingers used to be the bottleneck. Now it's your judgment.

**The endgame** is absurd on purpose. You're buying Dyson Spheres and Quantum Clusters. Numbers have twelve zeroes. The AI models cost more than countries.

And then you buy **The Singularity**. $500 trillion.

What happens next? Play and find out. The AI certainly did.

```
$ exit 0
```
